- **«Das ist die Frau, ___ Mann Arzt ist.» (cuya, fem.)** → deren · _Relativo genitivo fem./plural = deren (cuya/cuyos)._
- **«Die Kinder, ___ Eltern reich sind…» (plural)** → deren · _Plural relativo genitivo = deren._
- **«Die Leute, mit ___ ich arbeite…» (Dativo plural)** → denen · _Dativo plural del relativo = denen._
- **«Der Tisch, an ___ ich sitze…» (Dativo masc.)** → dem · _an + Dativo masc. → an dem._
- **«Das Buch, ___ ich lese…» (AKK neutro)** → das · _Relativo AKK neutro = das._
- **¿Cuál de estos verbos rige DATIVO?** → gefallen · _gefallen rige Dativo (Das Bild gefällt mir)._
- **«Das Auto gehört ___ Mann.» (a el hombre, Dat. masc.)** → dem · _gehören rige Dativo → dem Mann._
- **«Ich danke ___.» (a ti)** → dir · _danken rige Dativo → dir._
- **«Ich helfe ___.» (a ella)** → ihr · _helfen rige Dativo; sie(ella)→ihr._
- **¿Cuál rige Dativo?** → gratulieren · _gratulieren, danken, helfen, gehören… → Dativo._
- **«Das Essen schmeckt ___.» (a mí)** → mir · _schmecken rige Dativo → mir._
- **Comparativo de «teuer» (pierde la -e-):** → teurer · _teuer → teurer (cae la e: teu(e)rer)._
- **Comparativo de «groß»:** → größer · _groß → größer → am größten (irregular, sin -e en superl.)._
- **Comparativo de «viel»:** → mehr · _viel → mehr → am meisten._
- **Superlativo de «gern»:** → am liebsten · _gern → lieber → am liebsten._
- **Igualdad «tan rápido como»:** → so schnell wie · _Igualdad: so + adjetivo + wie._
- **«___ älter, desto erfahrener.» (cuanto más…)** → je · _Je + comparativo…, desto + comparativo…_
- **Adjetivo de 1 sílaba en comparativo (a/o/u):** → toma Umlaut (ä/ö/ü) · _Muchos monosílabos toman Umlaut: alt→älter, jung→jünger._
- **Superlativo de «nah»: am ___** → nächsten · _nah → näher → am nächsten (irregular)._
